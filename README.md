# DBSEE Agency - Premier Digital Marketing Agency

DBSEE is a high-performance, modern digital marketing agency platform built for high-growth businesses in Bahrain and beyond. This platform serves as the digital storefront and service hub for a premier agency specializing in social media management, advertising, video production, and business coaching.

## 🚀 Technology Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Lucide React](https://lucide.dev/) (Icons), [Framer Motion](https://www.framer.com/motion/) (Animations)
- **Database**: [MongoDB](https://www.mongodb.com/) (Form submissions & Data management)
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## ✨ Key Features

### 💎 Premium Visual Design

- **Modern Hero Sections**: Dynamic gradients, abstract SVG overlays, and stylish typography.
- **Responsive Architecture**: Seamless experience across mobile, tablet, and desktop devices.
- **Interactive UI**: Custom accordions, hover effects, and smooth transitions.

### 🛠️ Specialized Service Hubs

- **Social Media Management**: Strategic content planning and community engagement.
- **Social Media Advertising**: High-converting ad management for Meta and Google.
- **TikTok Ads**: Native creative strategies for the Bahraini market.
- **Video Production**: Professional galleries for Live Action, Event, Corporate, and Animated videos.
- **Coaching & Training**: Direct mentorship and business strategy consultation.

### 💼 Portfolio & Case Studies

- **Video Galleries**: Specialized grids for vertical (9:16) Reels and horizontal (16:9) video content.
- **Photography Masonry**: High-quality visual showcases with dynamic layouts.
- **Web Portfolio**: Clean displays for web development projects.

### 📈 Conversion Optimization

- **Lead Generation**: Integrated contact forms and "Request a Call" funnels.
- **Data Persistence**: Form submissions securely stored in MongoDB.
- **SEO Optimized**: Built-in best practices for search engine visibility.

## 📂 Project Structure

```bash
src/
├── app/              # Next.js App Router (Pages & API)
├── components/       # Modular UI & Layout components
│   ├── layout/       # Global Navbar, Footer, and HeroBanner
│   ├── services/     # Service-specific templates and sections
│   ├── portfolio/    # Reusable gallery and grid components
│   └── ui/           # Atomic UI elements (Buttons, Badges, etc.)
├── lib/              # Utility functions and database configuration
├── types/            # TypeScript interface definitions
└── public/           # Static assets and media
```

## 🛠️ Development

### Prerequisites

- Node.js 20.x or later
- MongoDB instance (local or Atlas)

### Getting Started

1. **Clone the repository**:

   ```bash
   git clone [repository-url]
   cd dbsee-agency
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```
   

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the project.
